# Binary Converter

## Description
This program converts numbers between decimal and binary formats.

Users can:
- Convert a decimal number to binary
- Convert a binary number to decimal

## Features 
-Handles positive and negative numbers
-Uses recursion for decimal to binary conversions 
-Simple command-line interface

## How to Run
1. Compile
   g++ binary_converter.cpp -o converter
2. Run:
   ./converter

## Technologies
-C++ 
-Standard Library (iostream, string, algorithm, cmath)
